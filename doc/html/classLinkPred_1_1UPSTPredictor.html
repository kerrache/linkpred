A link predictor that prestores edge scores. This allows to seemingly integrate results from external link prediction algorithms to LinkPred (for example, users may implement their own link prediction algorithm and then use this link predictor to use compare their results to algorithms available in LinkPred).

void LinkPred::UPSTPredictor< Network, EdgeRndIt, ScoreRndIt, EdgeRndOutIt >::loadEdgeScores ( std::string  fileName)

Load edge scores from file. This file should contain the scores of all non-exisitng links of net.

Parameters
fileNameThe name of the file containing edge scores. This should be a text file in which each line contains three columns separate by a space character (one or multiple spaces or tabs). The first two columns contain the the labels (not the internal IDs) of two nodes composing the edge. The third column contains the score. For example: A B 0.35 or: 5 8 2.6 All node labels that appear in this file must already exist in the network.
