Add an encoder-classifier link predictor.

Parameters
nameA unique identifier for the predictor.
encoderNameThe name of the encoder. Possible values are: DPW (DeepWalk), HMSM (Hidden Metric Space Model), LVS (LargeVis), LEM (Laplacian Eigenmaps), LIN (LINE), LLE (Locally Linear Embedding), MFC (Matrix Factorization), and N2V (Node2Vec).
classifierNameThe name of the classifier. Possible values are: FFN (feed-forward neural network withn default architecture), LSVM (linear SVM), LGR (logistic regression), NVB (naive Bayes). All classifiers except logistic regression requirte compilation with mlpack.
dimThe dimension of the embedding space. If set to zero, the default value is used (the default dimension depends on the ecnoder).
posRatioRatio of positive edges used in the training of the classifier.
negRatioRatio of negative edges used in the training of the classifier.
seedSeed of the random number generator.
